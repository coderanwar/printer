Peer-to-Peer Printing Using TCP/IP

Related Models:DCP‑L2540DW / DCP‑L2541DW / HL‑L2380DW / MFC‑L2700DW / MFC‑L2701DW / MFC‑L2703DW / MFC‑L2720DW / MFC‑L2740DW
In a Peer-to-Peer environment, each computer sends and receives data directly to and from each device. There is no central server controlling file access or printer sharing.
  1. Router
  2. Network Machine (your machine)
  • In a smaller network of two or three computers, we recommend the Peer-to-Peer printing method, as it is easier to configure than the Network Shared printing method.
  • Each computer must use the TCP/IP protocol.
  • The Brother machine needs an appropriate IP address configuration.
  • If you are using a router, the Gateway address must be configured on the computers and the Brother machine.
