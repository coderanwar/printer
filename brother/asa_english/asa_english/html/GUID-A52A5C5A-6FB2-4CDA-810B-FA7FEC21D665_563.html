Add Web Connect Shortcuts

Related Models:HL‑L2380DW / MFC‑L2720DW / MFC‑L2740DW
  • Web services may have been added and/or service names may have been changed by the provider since this document was published.
  • To add a Web Connect Shortcut, you need to have an account with the desired service.

    For more information, see the Web Connect Guide.

  1. Press (Shortcuts).
  2. Press where you have not added a Shortcut.
  3. Press or to display Web, and then press Web.
  4. If Information regarding the Internet connection is displayed, read the information and confirm it by pressing OK.
  5. Press or to display the available services, and then press the service you want.
  6. Press your account.

    If the account requires a PIN code, enter the PIN for the account using the LCD, and then press OK.

  7. Select the function you want to use.
    The functions that can be set vary depending on the chosen service.
  8. Read and confirm the displayed list of settings you have selected, and then press OK.
  9. Enter a name for the shortcut using the LCD, and then press OK.
  10. Press OK to save your Shortcut.
