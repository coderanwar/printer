Internet Fax (available as a download)

Internet Fax (I-Fax) allows you to send and receive fax documents using the Internet as the transport mechanism.

To use this feature, go to your model's Downloads page on the Brother Solutions Center at http://support.brother.com.

Before using this feature, configure the necessary machine's settings using the machine's control panel.
