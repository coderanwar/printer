Store Incoming Faxes in the Machine's Memory

Related Models:MFC‑L2700D / MFC‑L2701D / MFC‑L2700DW / MFC‑L2701DW / MFC‑L2703DW / MFC‑L2720DW / MFC‑L2740DW
Use the Fax Storage feature to store received faxes in the machine's memory. Retrieve your stored fax messages from a fax machine at another location using the Remote Retrieval commands. Your machine will print a backup copy of each stored fax.
MFC-L2700D/MFC-L2701D/MFC-L2700DW/MFC-L2701DW/MFC-L2703DW
MFC-L2720DW/MFC-L2740DW

MFC-L2700D/MFC-L2701D/MFC-L2700DW/MFC-L2701DW/MFC-L2703DW

You must turn on the Memory Receive option to use this function.
  1. Press Menu.
  2. Press or to display the Fax option, and then press OK.
  3. Press or to display the Remote Fax Opt option, and then press OK.
  4. Press or to display the Forward/Store option, and then press OK.
  5. Press or to display the Fax Storage option, and then press OK.
  6. Press Stop/Exit.

MFC-L2720DW/MFC-L2740DW
