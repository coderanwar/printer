Configure the Brother Machine for Email or I-Fax

Related Models:MFC‑L2720DW / MFC‑L2740DW

Before using I-Fax, configure your Brother machine to communicate with your network and mail server. You can configure these items using the machine's control panel, Web Based Management, Remote setup, or BRAdmin Professional 3. Make sure the following items are configured on your machine:

  • IP address (If you are already using your machine on your network, the machine's IP address has been configured correctly.)
  • Email address
  • SMTP, POP3, IMAP4 server address/port/Authentication method
  • Mailbox name and password

If you are unsure of any of these items, contact your system administrator.
