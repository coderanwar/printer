Save Copy Options as Shortcuts

Related Models:HL‑L2380DW / MFC‑L2720DW / MFC‑L2740DW
Store the copy options and settings you use frequently by saving them as Shortcuts.
  1. Press (Copy)>Options.
  2. Press or to display the options, and then press the setting you want to change. Repeat this step for each option and setting you want to change.
  3. When finished, press Save as Shortcut.
  4. Read and confirm the displayed list of options you have chosen for the Shortcut, and then press OK.
  5. Enter a name for the Shortcut using the LCD, and then press OK.
  6. Press OK to save your Shortcut.
