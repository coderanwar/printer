What is BRAdmin Light? (Windows® and Macintosh)

Related Models:DCP‑L2540DW / DCP‑L2541DW / HL‑L2380DW / MFC‑L2700DW / MFC‑L2701DW / MFC‑L2703DW / MFC‑L2720DW / MFC‑L2740DW

BRAdmin Light is a utility for the initial setup of Brother network-connected devices. This utility can search for Brother products on your network, view the status and configure the basic network settings, such as IP address.

Go to your model's Downloads page on the Brother Solutions Center at http://support.brother.com to download BRAdmin Light.

  • If you are using Windows® Firewall, or the firewall function of an anti-spyware or antivirus application, temporarily disable them. When you are sure you can print, configure the software settings following the instructions.
