Send Tone Signals

Related Models:MFC‑L2700D / MFC‑L2701D / MFC‑L2700DW / MFC‑L2701DW / MFC‑L2703DW / MFC‑L2720DW / MFC‑L2740DW
If you have a Pulse dialling service, but must send Tone signals (for example, for telephone banking), follow these steps. If you have Touch Tone service, you will not need this feature to send tone signals. (This feature is not available in New Zealand.)
MFC-L2700D/MFC-L2701D/MFC-L2700DW/MFC-L2701DW/MFC-L2703DW
MFC-L2720DW/MFC-L2740DW

MFC-L2700D/MFC-L2701D/MFC-L2700DW/MFC-L2701DW/MFC-L2703DW

  1. Press (FAX).
  2. Press Hook.
  3. Press # on the machine’s control panel.

    Any digits dialled after this will send tone signals.

When you are finished with the call, the machine will return to the Pulse dialling service.

MFC-L2720DW/MFC-L2740DW

  1. Press (Fax).
  2. Press Hook.
  3. Press # on the machine's LCD.

    Any digits dialled after this will send tone signals.

When you are finished with the call, the machine will return to the Pulse dialling service.
