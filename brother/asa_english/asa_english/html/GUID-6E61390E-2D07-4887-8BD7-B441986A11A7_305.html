Configure the Network PC-FAX Receiving Settings (Windows®)

Related Models:MFC‑L2700DW / MFC‑L2701DW / MFC‑L2703DW / MFC‑L2720DW / MFC‑L2740DW
  • If you installed the machine following the instructions for a network user, then the machine should already be configured for your network.
  1. Right-click the (PC-FAX Receive) icon in your computer task tray, and then select the PC-Fax Rx Setup option.
    The PC-Fax Rx Setup dialog box appears.
  2. Click the Network Settings button.
    The Network Settings dialog box appears.
  3. Select the appropriate connection method.
    • Specify your machine by address
      Type the machine's IP address in the IP Address field.
    • Specify your machine by name
      Type the machine's node name in the Node Name field, or click the Browse button and select the Brother machine you want to use from the list. Click OK.
  4. To change the computer name that will appear on the machine's LCD, type the new name in the Display Your PC Name (Max 15 characters) field.
  5. Click OK.
