Set Auto Power Off Mode

Related Models:DCP‑L2520D / DCP‑L2540DW / DCP‑L2541DW / HL‑L2380DW
If the machine is in Deep Sleep Mode for certain length of time, based on your model and settings, the machine will go into Power Off Mode automatically. The machine does not go into Power Off Mode when the machine is connected to a wired or wireless network or has secure print data in the memory.
  • To start printing, press on the control panel and then send a print job.
