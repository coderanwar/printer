MDN

The Message Disposition Notification (MDN) field of the control panel menu requests the status of the Internet Fax/E-mail message after delivery through the Send Mail Transfer Protocol (SMTP) transport system.

Once the message has arrived at the receiver, this data is used when the machine or user reads or prints the received Internet Fax or E-mail.

For example, if the message is opened for reading or is printed, the receiver sends back a notification to the original sending machine or user.

The receiver must support the MDN field to be able to send a notification report, otherwise the request will be ignored.
