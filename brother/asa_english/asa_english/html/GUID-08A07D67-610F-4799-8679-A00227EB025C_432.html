Security

  • Lock the Machine Settings
    Before turning on the machine's access lock feature, make a careful note of your password. If you forget the password, you must reset all passwords stored in the machine by calling your administrator or Brother customer service.
  • Before Using Network Security Features
    Your Brother machine employs some of the latest network security and encryption protocols available today. These network features can be integrated into your overall network security plan to help protect your data and prevent unauthorized access to the machine.
  • Secure Function Lock 3.0
    Brother's Secure Function Lock 3.0 increases security by restricting the functions available on your Brother machine.
