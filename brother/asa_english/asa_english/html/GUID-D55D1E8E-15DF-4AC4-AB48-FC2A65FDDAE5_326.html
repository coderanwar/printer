What is Remote Setup? (Windows® and Macintosh)

Related Models:MFC‑L2700DW / MFC‑L2701DW / MFC‑L2703DW / MFC‑L2720DW / MFC‑L2740DW

Remote Setup is a program for configuring many machine and network settings from either a Windows® or a Macintosh application. When you start this application, the settings on your machine are automatically downloaded to your computer and displayed on your computer screen. If you change the settings, you can upload them directly to the machine.

This utility can be used for either a USB connection or a network connection.
