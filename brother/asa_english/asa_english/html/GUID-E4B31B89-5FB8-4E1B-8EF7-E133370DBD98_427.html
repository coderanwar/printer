Setup Relay

Related Models:MFC‑L2720DW / MFC‑L2740DW

Relay Broadcast

Allows the machine to receive a document over the Internet and then relay it to other fax machines through conventional analog landlines.

Relay Domain

Register up to ten Domain Names that are allowed to request a Relay Broadcast.

Relay Report

A Relay Broadcast Report is a list of any relayed broadcasts that have been sent through the machine. It can be printed at the machine that will act as a Relay Station for all Relay Broadcasts.

To use this function, you must assign the relay domain in the "Trusted Domains" section of the Relay function settings.
