Send a Scanned Image as a Fax Using ControlCenter4 Home Mode (Windows®)

Related Models:MFC‑L2700D / MFC‑L2701D / MFC‑L2700DW / MFC‑L2701DW / MFC‑L2740DW
Use the Send Scanned Image button in ControlCenter4 to scan a page or a document and fax it from your computer using Brother PC-FAX.
  • Select Home Mode as the mode setting for ControlCenter4.

  • PC-FAX can send only Monochrome fax documents.

  1. Click the (ControlCenter4) icon in the task tray, and then click Open.
  2. Click the PC-FAX tab.
  3. Click the Send Scanned Image button.
  4. Configure the scan settings, if needed.
  5. Click (Scan).
  6. Confirm and edit (if needed) the scanned image in the image viewer.
  7. Click (Send).
    The Brother PC-FAX dialog box appears.
  8. Type a fax number, and then click Start.
