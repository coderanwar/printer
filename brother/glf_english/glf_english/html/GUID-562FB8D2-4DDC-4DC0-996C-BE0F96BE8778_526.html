Change the Default Mode

Related Models:DCP‑L2520D / DCP‑L2540DW / DCP‑L2541DW / MFC‑L2700D / MFC‑L2701D / MFC‑L2700DW / MFC‑L2701DW
When the machine is idle or you press Stop/Exit, the machine will return to the mode you set.
  • For MFC models, the machine has FAX, SCAN and COPY modes. Factory setting is FAX mode.
  • For DCP models, the machine has Scan and Copy modes. The Copy mode is fixed default mode.
  1. Press Menu.
  2. Press or to display the General Setup option, and then press OK.
  3. Press or to display the Default Mode option, and then press OK.
  4. Press or to display the Fax, Copy, or Scan option, and then press OK.
  5. Press Stop/Exit.
