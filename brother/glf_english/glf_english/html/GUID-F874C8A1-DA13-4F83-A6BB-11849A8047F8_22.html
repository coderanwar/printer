Paper Settings

  • Change the Paper Size
    When you change the size of the paper you load in the tray, you must change the paper size setting on the LCD at the same time.
  • Change the Paper Type
    To get the best print quality, set the machine for the type of paper you are using.
  • Set the Check Size
    Your Brother machine's Check Size setting causes the machine to display a message when you remove the paper tray from the machine, asking if you changed the paper size.
