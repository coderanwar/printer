Network Menu on the Brother Control Panel

Related Models:DCP‑L2540DW / DCP‑L2541DW / HL‑L2380DW / MFC‑L2700DW / MFC‑L2701DW / MFC‑L2740DW
The control panel's Network menu selections allow you to configure the Brother machine for your network.

Note that your Brother machine includes the BRAdmin Light, Web Based Management, and Remote Setup applications, which you can use to configure many aspects of the network.

Remote Setup is not available for DCP models.
