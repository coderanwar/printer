Ethernet (Wired Network Only)

Related Models:DCP‑L2540DW / DCP‑L2541DW / HL‑L2380DW / MFC‑L2700DW / MFC‑L2701DW / MFC‑L2740DW

Ethernet link mode allows the print server to operate in 100BASE-TX full or half duplex or in 10BASE-T full or half duplex mode by auto-negotiation.

If you set this value incorrectly, you may not be able to communicate with your print server.
