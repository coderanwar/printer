Scan to E-mail

Related Models:DCP‑L2540DW / DCP‑L2541DW / HL‑L2380DW / MFC‑L2700DW / MFC‑L2701DW / MFC‑L2740DW

Select Scan to E-mail to change the resolution and file type for an email attachment. You can scan and send a monochrome or colour document from the computer as an attachment.
