Use Dual Access to Dial, Scan, Send, and Receive Faxes Simultaneously

Related Models:MFC‑L2700D / MFC‑L2701D / MFC‑L2700DW / MFC‑L2701DW / MFC‑L2740DW
Dial a number and scan the fax into memory - even while the machine is sending from memory, receiving faxes, or printing from a computer.

The LCD shows the new job number.

The number of pages you can scan into the machine's memory varies depending on the type of data.

  • If an Out of Memory message appears while scanning the document, press the button shown in the table to send the scanned pages, or to cancel the operation.
