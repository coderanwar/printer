Print a Fax Stored in the Machine's Memory

Related Models:MFC‑L2700D / MFC‑L2701D / MFC‑L2700DW / MFC‑L2701DW / MFC‑L2740DW
If you select Fax Storage, you can still print a fax from the memory when you are at your machine.
MFC-L2700D/MFC-L2701D/MFC-L2700DW/MFC-L2701DW
MFC-L2740DW

MFC-L2700D/MFC-L2701D/MFC-L2700DW/MFC-L2701DW

You must turn on the Memory Receive option to use this function.
  1. Press Menu.
  2. Press or to display the Fax option, and then press OK.
  3. Press or to display the Remote Fax Opt option, and then press OK.
  4. Press or to display the Print Document option, and then press OK.
  5. Press Start.
  6. Press Stop/Exit.

MFC-L2740DW

  1. Press (Settings)>All Settings>Fax>Print Document.
  2. Press OK.
  3. Press .
