View and Print Faxes Using ControlCenter4 (Windows®)

Related Models:MFC‑L2700D / MFC‑L2701D / MFC‑L2700DW / MFC‑L2701DW / MFC‑L2740DW
Use ControlCenter4's View Received PC-FAX button to view and print received fax messages using Brother's PC-FAX software.
  • The PC-FAX software can receive only monochrome fax documents.
  1. Click the (ControlCenter4) icon in the task tray, and then click Open.
  2. Click the PC-FAX tab.
  3. Click the View Received PC-FAX button.
  4. The PC-FAX receiving feature is activated and starts to receive faxes from your machine. The received faxes appear. To view the faxes, select the faxes, and then click (View).
  5. To print the received faxes, select the faxes, and then click (Print).
  6. Select the print settings, and then click (Start Printing).
