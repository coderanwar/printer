Node Name

The Node Name is a machine name on the network. This name is often referred to as a NetBIOS name; it will be the name that is registered by the WINS server on your network. Brother recommends the name "BRNxxxxxxxxxxxx" for a wired network or "BRWxxxxxxxxxxxx" for a wireless network (where "xxxxxxxxxxxx" is your machine's MAC address / ethernet address.)
