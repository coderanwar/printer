Authentication

Most wireless networks use some kind of security settings. These security settings define the authentication (how the device identifies itself to the network) and encryption (how the data is encrypted as it is sent on the network). If you do not correctly specify these options when you are configuring your Brother wireless machine, it will not be able to connect to the wireless network. Therefore, care must be taken when configuring these options.
Authentication Methods for a Personal Wireless Network
A personal wireless network is a small network (such as a home wireless network) without IEEE 802.1x support.
  • Open system

    Wireless devices are allowed to access the network without any authentication.

  • Shared key

    A secret pre-determined key is shared by all devices that will access the wireless network. The Brother wireless machine uses the WEP key as the pre-determined key.
