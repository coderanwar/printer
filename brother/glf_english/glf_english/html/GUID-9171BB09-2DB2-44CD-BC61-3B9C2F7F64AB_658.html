BRPrint Auditor (Windows®)

Brother's BRPrint Auditor software brings the monitoring power of Brother network management tools to locally-connected machines. This utility allows a client computer to collect usage and status information from your Brother machine connected via the parallel or USB interface. The BRPrint Auditor can then pass this information to another computer running BRAdmin Professional 3 on the network. This allows the administrator to check items, such as page counts, toner and drum status, and the firmware version. In addition to reporting to Brother network management applications, this utility can email the usage and status information directly to a predefined email address in a CSV or XML file format (SMTP Mail support required). The BRPrint Auditor utility also supports email notification for reporting warning and error conditions.
