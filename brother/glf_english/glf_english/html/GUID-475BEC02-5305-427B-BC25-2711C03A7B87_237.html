Receive Faxes into Memory When the Paper Tray is Empty

Related Models:MFC‑L2700D / MFC‑L2701D / MFC‑L2700DW / MFC‑L2701DW / MFC‑L2740DW

As soon as the paper tray becomes empty during fax reception, the machine begins receiving faxes into the machine’s memory.

Incoming faxes will be stored until the machine's memory becomes full or until paper is loaded in the paper tray. When the memory is full, the machine will stop automatically answering calls. To print the faxes, load paper in the tray.

(2-Line LCD models)

If you turn off the Memory Receive option, the machine will stop answering subsequent calls after running out of paper.
