Print on Both Sides of the Paper (Macintosh)

  • Select A4 sized paper when using the automatic 2-sided printing function.
  • Make sure the back cover is closed.
  • If paper is curled, straighten it and put it back in the paper tray.
  • Use regular paper or thin paper. DO NOT use bond paper.
  • If the paper is too thin, it may wrinkle.
  • When you use the manual 2-sided printing function, it is possible there may be paper jams or poor print quality.
  1. From an application, such as Apple TextEdit, click the File menu, and then select Print.
  2. Select Brother MFC-XXXX series (where XXXX is your model name).

    For more page setup options, click the Show Details button.

  3. Select the Orientation option that matches the way you want the document to print.
  4. For automatic 2-sided printing, do the following:
    1. Click the application pop-up menu, and then select Layout.
      The Layout setting options appear.
    2. In the Two-Sided section, select the Short-Edge binding or Long-Edge binding option.
    3. Change other printer settings, if needed.
    4. Click Print.
  5. For manual 2-sided printing, do the following:
    1. Click the application pop-up menu, and then select Paper Handling.
      The Paper Handling options appear.
    2. In the Pages to Print pop-up menu, select Even Only.
    3. Change other print settings, if needed.
    4. Click the Print button.
    5. After the machine prints the even-numbered pages, remove the printed pages from the output paper tray.
    6. Make sure the pages are completely flat, and put the paper back in the tray, blank side facing down.
    7. Repeat steps 1-3, selecting the same printer and settings you used to print the even-numbered pages.
    8. Click the application pop-up menu, and then select Paper Handling. The Paper Handling options appear.
    9. In the Pages to Print pop-up menu, select Odd Only.
    10. Click Print.
