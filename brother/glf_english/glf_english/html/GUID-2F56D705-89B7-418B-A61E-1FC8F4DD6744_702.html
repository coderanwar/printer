LLMNR

The Link-Local Multicast Name Resolution (LLMNR) protocol resolves the names of neighboring computers, if the network does not have a Domain Name System (DNS) server. The LLMNR Responder feature works in both the IPv4 or IPv6 environment when using a computer that has the LLMNR Sender feature, such as Windows Vista® and Windows® 7.
