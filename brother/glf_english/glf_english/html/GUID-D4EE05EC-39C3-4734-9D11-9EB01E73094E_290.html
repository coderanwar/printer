Send a Fax from ControlCenter4 Using PC-FAX (Windows®)

Related Models:MFC‑L2700D / MFC‑L2701D / MFC‑L2700DW / MFC‑L2701DW / MFC‑L2740DW
Use the Send Image File button in ControlCenter4 to fax an image file from your computer using Brother PC-FAX.

  • PC-FAX can send only Monochrome fax documents.

  1. Click the (ControlCenter4) icon in the task tray, and then click Open.
  2. Click the PC-FAX tab.
  3. Click the Send Image File button.
  4. Click on the folder tree to browse for the folder where the image you want to send is located. Select the file you want to send.
  5. Click (Send).
    The Brother PC-FAX dialog box appears.
  6. Type a fax number, and then click Start.
