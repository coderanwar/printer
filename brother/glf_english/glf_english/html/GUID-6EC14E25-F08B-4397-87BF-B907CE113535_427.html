About Using TX Lock

Related Models:MFC‑L2700D / MFC‑L2701D / MFC‑L2700DW / MFC‑L2701DW
Block unauthorized data transmission to and from the machine.

While TX Lock is on, the following operations are available IF they active before you turned on TX Lock:

  • Receive faxes
  • Fax Forward
  • Paging
  • Remote Retrieval
  • PC Fax Receive
    To print the faxes in memory, turn off TX Lock.

While TX Lock is on, the following operations are NOT available:

  • Fax send
  • Copy
  • PC print
  • Scan
