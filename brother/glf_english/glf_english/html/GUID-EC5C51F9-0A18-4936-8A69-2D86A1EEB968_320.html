Install or Access Network Management Software and Utilities

Related Models:DCP‑L2540DW / DCP‑L2541DW / HL‑L2380DW / MFC‑L2700DW / MFC‑L2701DW / MFC‑L2740DW
Before using your Brother machine in a network environment, you must install the Brother software and configure the appropriate TCP/IP network settings on the machine itself.

We recommend using the Brother installer to install the Brother software and guide you through the software and network installation.

If you do not want to or are unable to use the Brother installer, see Related Information.
